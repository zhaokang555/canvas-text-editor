# canvas-text-editor

A rich text editor based on HTML Canvas. 

基于 HTML Canvas 的富文本编辑器.

## Online Demo

https://zhaokang555.github.io/canvas-text-editor/

## Environment Setup / 开发环境搭建

Run
```
npm install
npm run dev
```

Then open `http://localhost:3000/canvas-text-editor/`.
